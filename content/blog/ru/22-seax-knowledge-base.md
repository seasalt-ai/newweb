# Традиционная база знаний

По сути, база знаний (БЗ) — это просто библиотека (в идеале) хорошо организованной и легкодоступной информации, которая используется в режиме самообслуживания онлайн. Хорошие системы баз знаний будут иметь такие функции, как иерархическая организация контента, поиск и маркировка, чтобы помочь пользователям легче находить правильную информацию.

Поддержание подробной базы знаний является стандартной практикой для большинства компаний в настоящее время. Будь то помощь сотрудникам в обмене внутренней информацией о своем продукте, ответы на вопросы потенциального клиента, помощь клиентам в устранении неполадок или все вышеперечисленное — обеспечение доступа к ключевой информации для сотрудников и клиентов означает более эффективную работу и более высокую удовлетворенность клиентов.

Обычно база знаний реализуется и поддерживается с помощью системы управления контентом или системы управления знаниями. Эти системы могут различаться по масштабу в зависимости от потребностей организации, начиная от простого менеджера документов до многофункционального сервиса, который включает рабочие процессы публикации, таргетинг аудитории, инструменты для совместной работы и многое другое. Хотя эти системы универсальны по-разному, они почти всегда предназначены для доступа через взаимодействие с веб-страницей или приложением. Для конкретного случая использования агента по обслуживанию клиентов (который обычно использует базу знаний как один из своих основных ресурсов при оказании помощи клиентам) необходима тесная интеграция с программным обеспечением контакт-центра, чтобы агенты могли обрабатывать запросы пользователей максимально бесшовно.

# База знаний SeaX

Наша база знаний была разработана с первого дня с учетом очень конкретного варианта использования: голосового обслуживания клиентов. Хотя большинство, если не все, существующие системы баз знаний полагаются на навигацию по иерархическим веб-страницам или ввод поискового запроса, наша база знаний должна была быть быстрее и независимее, чтобы представители службы поддержки клиентов могли уделять все свое внимание клиенту, при этом быстро отвечая на вопросы.

## Быстрый и точный поиск

<img src="/images/blog/22-seax-knowledge-base/kb-manual-search.png" alt="Результаты ручного поиска в базе знаний SeaX."/>

На самом базовом уровне наша база знаний работает на невероятно быстром и точном поисковом движке. Мы используем самые современные методы обработки естественного языка и извлечения информации для получения смысла из обычного текста, примеров запросов и вспомогательных URL-адресов, а также для сопоставления высказываний клиентов с наиболее релевантными записями базы знаний. Движок базы знаний очень расширяем и может поддерживать миллиарды документов без заметного изменения времени ответа.

## Автоматические предложения в реальном времени

Но то, что мы показали до сих пор, — это все еще ручной поиск. Живые агенты заняты взаимодействием с клиентами и будут терять драгоценное время, вводя ручной поиск в базу знаний каждый раз, когда им нужна информация. По этой причине наибольшую ценность, которую приносит база знаний SeaX, представляет собой автоматический поиск в реальном времени как для текстовых, так и для голосовых взаимодействий.

<center>
<img src="/images/blog/22-seax-knowledge-base/kb-automatic-search.png" alt="SeaX KB показывает автоматические предложения статей для входящего сообщения пользователя."/>

*SeaX KB показывает автоматические предложения статей для входящего сообщения пользователя.*
</center>

Каждый раз, когда поступает новое сообщение от пользователя, база знаний будет автоматически запрашиваться с использованием точного сообщения клиента. В режиме реального времени, пока клиент разговаривает, агенту будут предоставляться актуальные предложения статей из базы знаний для его справки.

И это работает и с голосовыми вызовами! Наш последний пост в блоге, [Придайте своему контакт-центру собственный голос с помощью SeaX Voice Intelligence](https://seasalt.ai/blog/21-seax-voice-intelligence/), продемонстрировал передовой движок Seasalt.ai для преобразования речи в текст. Платформа SeaX использует этот движок для транскрибирования всех голосовых вызовов в реальном времени. В результате мы можем использовать эти транскрипции для различных последующих приложений, включая автоматический поиск в базе знаний.

## Шаблоны ответов

<center>
<img src="/images/blog/22-seax-knowledge-base/kb-response-template.png" alt="Агент отвечает клиенту одним щелчком мыши, используя сгенерированный шаблон ответа SeaX KB."/>

*Агент отвечает клиенту одним щелчком мыши, используя сгенерированный шаблон ответа SeaX KB.*
</center>

Результаты поиска из базы знаний имеют одну дополнительную функцию, которая помогает ускорить ответы агентов для текстовых взаимодействий. Когда агент находит соответствующую статью базы знаний, он может просто щелкнуть значок `+` слева от заголовка, чтобы вставить шаблон ответа в свое окно чата. В бэкэнде, каждый раз, когда база знаний ищется, она генерирует письменный ответ на вопрос пользователя на основе наиболее релевантной информации в предложенной статье базы знаний и включает любые вспомогательные ссылки. Это может значительно улучшить время ответа агента, поскольку агент больше не начинает с чистого листа. Вместо этого у него уже есть важная информация из статьи базы знаний в окне чата, поэтому он может просто отредактировать ее и отправить.

Теперь, когда мы увидели, что может делать движок базы знаний, остается один насущный вопрос о бэкэнде: как управлять информацией в базе знаний? Платформа SeaX имеет полностью интегрированный пользовательский интерфейс управления базой знаний, доступный администраторам со страницы настроек.

*Интерфейс управления базой знаний SeaX.*

Интерфейс также поддерживает редактирование и удаление записей базы знаний, чтобы вы могли постоянно поддерживать свою базу знаний в актуальном состоянии.
